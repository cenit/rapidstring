Copies characters to a stack string.

Overwrites any existing data.

Parameters
[in,out]sAn initialized stack string.
[in]inputThe input to assign to s.
[in]nThe length of input.
Warning
The input length must be smaller or equal to RS_STACK_CAPACITY. If this is inconvenient for your usage, use rs_cpy().
Allocation
Never.
Complexity
Linear in n.
